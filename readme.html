Dynamic Web TWAIN JavaScript Sample

This project utilizes the Dynamic Web TWAIN SDK. This sample demonstrates the following Web TWAIN functionalities:

  • Scan documents from physical scanners
  • Load documents from local disk (bmp/jpg/png/tif/pdf)
  • View and process the documents already scanned/loaded
    • Editing (rotate/flip/mirror/crop, etc.)
    • Saving or uploading

Basic information

Programming languages: JavaScript

Dynamic Web TWAIN Version: 19.x

This sample is an implementation of Dynamic Web TWAIN, a JavaScript-based Software Development Kit (SDK) that enables document scanning directly into web browsers via front-end APIs. For your evaluation, this web application combines a user interface and sample back-end code so you can evaluate the document upload flow. The back-end code samples are strictly for demonstration purposes and are not part of the SDK.

For your convenience, a 14-day free trial key has been embedded in this sample project. If you need additional time to evaluate the SDK, please request a trial extension.

If you have any questions, comments, or concerns, please contact us.

How to run the sample

  1. Copy the entire "Sample" directory, which is unzipped from the downloaded file, to the web server.

  2. Navigate to http://{server-path}/Sample/index.html in your browser to access the sample.

Note: This sample uploads scanned images to a Dynamsoft server, where the backend code is deployed to receive data from the Dynamic Web TWAIN client-side code.

How to deploy your own upload server

You can also upload scanned documents to your own server. We provide backend code to help you get started. You can find the code download links and deployment guides at the following links.
